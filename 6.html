Common Tags Part 2 and Attributes

Attributes

Attributes provide additional information about HTML elements. They usually Usually comes in name-value pairs.
Example: <img src="./assets/images/logo.jpg">

<meta>

  • Defines metadata about an HTML document.
  • Metadata is used by browsers, search engines, and other web services.
  • Always go inside the element.

<charset>

  • Specifies the character set used in the page.

<abbr>

  • Defines an abbreviation or an acronym.
  • Use the title attribute to show the expanded form of the abbreviation/ acronym.

<img>

  • Used to embed an image.
  • Has NO closing tag.

<form>

  • Creates a form for user input.

<label>

  • Defines a label for elements inside a form.

<input>

  • Specifies an input field where the user can enter data.

<textarea>

  • Defines a multi-line text input control.

<select>

  • Creates a drop-down list.

<option>

  • Defines an option in a select list.

<button>

  • Defines a clickable button.

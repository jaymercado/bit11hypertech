Common Tags Part 1

<hr>

  • Horizontal line across the width of the text.
  • Has NO closing tag.

<strong>

  • Defines important text.

<br>

  • Single line break.

<q>

  • Quote a short passage.

<cite>

  • Defines the title of a creative work.

<em>

  • Defines emphasized text.

<address>

  • Defines a contact information, like email address, URL, physical address, phone number, social media, etcâ€¦

<details>

  • Specifies additional details.
  • Creates an interactive widget that the user can open and close.

<summary>

  • Defines a visible heading for the <details> element.

<ul>

  • Defines an ordered list.

<ul>

  • Defines an unordered list.

<li>

  • Defines an list item.

<table>

  • Defines a table.

<caption>

  • Gives a title to a table.
  • Used as the first child element of a table.

<thead>

  • Groups header content in a table.

<tbody>

  • Groups body content in a table.

<tfoot>

  • Groups body content in a table.

<tr>

  • Defines a row in a table.

<th>

  • Defines a header cell in a table.

<td>

  • Defines a standard data cell in a table.

Basic Web Page Structure

<!DOCTYPE html>

  • Document Type Declaration.
  • Has NO closing tag.
  • Tells the browser what type of file and what version of HTML to expect.

<html>

  • The root element.
  • Wraps around all other elements (except the doctype).

<head>

  • Where information that does not appear in the body of the work is put.
  • Contains metadata (data about data).

<body>

  • Contains all the visible content of the page.
  • There can be only one <body> element per document.

<title>

  • Defines the title of the document.
  • Required in all HTML documents.
  • Very important for search engine optimization (SEO).

<h1> to <h6>

  • Used to define HTML headings.
  • <h1> defines the most important heading.
  • <h6> defines the least important heading.
  • Can be used as the title of the document or of a section.

<p>

  • Represents a paragraph.

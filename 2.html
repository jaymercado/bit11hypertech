The Request/Response Cycle

Definition of Terms:

  • Servers: Machines that hold shared resources.
  • Clients: Devices for personal use.
  • Uniform Resource Locator:
    1. Protocol : Set of rules or standards that must be followed when sending requests and responses.
    2. Domain : Distinguishes the entity (device) you want to connect to. It is associated to an Internet Protocol (IP) address. An IP address is a numerical value that identifies a device connected to the internet.
    3. Path : Refers to the exact location of a page, post, file, or other asset.
  • Domain Name System: Maps domain names to numeric IP addresses.

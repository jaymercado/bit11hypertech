HTML5

HTML stands for Hyper Text Markup Language. It is used to build the structure of web pages. It was invented by Tim Bernes-Lee. Today we are using the 5th generation of HTML.

HTML Element

An HTML document is made up of HTML elements, which are components nested one inside the other. An HTML element is composed of:
  • Opening Tage
  • Content
  • Closing Tage
Example: <p>Hello, World!</p>

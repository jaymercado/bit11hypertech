CSS

CSS stands for Cascading Style Sheets. It is used to describe the presentation/ style of an HTML document pages. Today we are using the 3rd generation of CSS. There are three methods of implementing styling information to an HTML document.

  1. Inline styles — Using the style attribute in the HTML start tag.
    Example: <h1 style=“color: green;”>Hello, World</h1>
  2. Embedded style — Using the <style> element in the head section of the document.
    Example: <head><style>h1{color: green;}<style></head>
  3. External style sheet — Using the <link> element, pointing to an external CSS files.
    Example: <head><link rel="stylesheet" href="styles.css"></head>
